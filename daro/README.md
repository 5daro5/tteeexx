* Beauty Salon Web Application

Веб-приложение для салона красоты с возможностью просмотра товаров, регистрации, авторизации и оставления отзывов.

* Описание проекта

Проект представляет собой полнофункциональное веб-приложение, состоящее из backend API на Node.js/Express и frontend части на HTML/CSS/JavaScript. Приложение позволяет пользователям просматривать каталог товаров, регистрироваться, авторизоваться и оставлять отзывы на товары.

**Основные функции:**
- Регистрация и авторизация пользователей
- Просмотр каталога товаров
- Фильтрация и поиск товаров
- Оставление отзывов с оценкой от 1 до 5 звезд
- Личный кабинет со статистикой отзывов
- Просмотр отзывов других пользователей

* Установка и запуск

* 1. Клонирование репозитория
```bash
git clone https://gitverse.ru/Belisarius/Web-Application.git
cd Web-Application

* 2. Установка зависимостей backend
```bash
cd backend
npm install

* 3. Запуск сервера
```bash
node server.js
Сервер запустится на порту 3000. В консоли должно появиться:
"База данных подключена успешно"
"Таблицы созданы успешно"
"Сервер запущен на порту 3000"

* 4. Запуск клиентской части
Откройте любой HTML файл в корневой папке:
index.html - главная страница
products.html - каталог товаров
login.html - страница входа и регистрации
dashboard.html - личный кабинет (доступен после авторизации)

##Примеры использования
Регистрация нового пользователя
Откройте login.html
Нажмите "Создать аккаунт"
Заполните форму (имя, email, пароль)
Нажмите "Создать аккаунт"

Просмотр товаров и отзывов
Откройте products.html
Используйте поиск для фильтрации товаров
Нажмите "Отзывы" на карточке товара

Оставление отзыва (только для авторизованных)
Авторизуйтесь на сайте
На странице товаров нажмите "Оставить отзыв"
Напишите отзыв и поставьте оценку

##Технические требования
Node.js: версия 18.x или выше
npm: версия 9.x или выше
Операционная система: Windows, macOS, Linux
База данных: SQLite (встроенная)

Таблицы базы данных
users (пользователи)
id - уникальный идентификатор
name - имя пользователя
email - email (уникальный)
password - пароль
created_at - дата регистрации

products (товары)
id - уникальный идентификатор
name - название товара
price - цена
description - описание
image_url - ссылка на изображение
category - категория
created_at - дата добавления

reviews (отзывы)
id - уникальный идентификатор
user_id - ID пользователя
product_id - ID товара
review - текст отзыва
stars - оценка (1-5)
created_at - дата отзыва

shops (магазины)
id - уникальный идентификатор
address - адрес магазина
phone - телефон
latitude - широта
longitude - долгота
created_at - дата добавления
